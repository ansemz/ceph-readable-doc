cephfs-top – Ceph Filesystem Top Utility¶

提纲¶

cephfs-top [flags]

描述¶

cephfs-top 为 Ceph 文件系统提供了 top(1) 风格的功能。 可以显示各种客户端指标并实时更新。

Ceph 元数据服务器周期性地向 Ceph 管理器发送客户端指标。 Ceph 管理器的 Stats 插件有接口获取这些指标。

选项¶

--cluster¶

集群：要连接的 Ceph 集群，默认是 ceph 。

--id¶

Id: 用于连接 Ceph 集群的客户端，默认是 fstop 。

--selftest¶

执行一次自检。这个模式对 stats 模块执行一次健全性检查。

字段描述¶

chit

cap hit rate

rlat

读延时

wlat

写延时

mlat

元数据延时

dlease

dentry lease rate

ofiles

已打开文件的数量

oicaps

number of pinned caps

oinodes

打开的 inode 数量

rtio

读 IO 的总量

wtio

写 IO 的总量

raio

读 IO 的平均尺寸

waio

写 IO 的平均尺寸

rsp

与上次刷新相比的读 IO 速度

wsp

与上次刷新相比的写 IO 速度

使用范围¶

cephfs-top 是 Ceph 的一部分，这是个伸缩力强、开源、 分布式的存储系统，更多信息参见 https://ceph.com/ 。
