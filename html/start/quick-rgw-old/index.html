Ceph 对象存储速读¶

要使用 Ceph 对象存储快速入门，你必须先完成存储集群快速入门指南，并且至少有一个 RGW 例程在正常运行。

配置新的 RGW 例程¶

按照存储集群快速入门创建的 RGW 例程会用嵌入式的 CivetWeb 网页服务器运行。 ceph-deploy 会创建此例程、并用默认参数自动启动它。

关于 RGW 例程的管理，请参考 RGW 管理手册。

其他细节可以在 Ceph 对象网关的配置手册里找到，但是特定于 Apache 的步骤不再需要了。

Note

用 ceph-deploy 部署 RGW 、并用 CivetWeb 网页服务器替代 Apache 是从 Hammer 版起才有的新功能。
